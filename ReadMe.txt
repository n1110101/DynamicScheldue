Задание по спецкурсу "Введение в распределённые системы и сети"

Автор программы:
	Титов Николай, 216 группа

Содержание данного архива:
	task.cpp - файл с реализацией задачи,
	output.xml - XML-файл с результатом вычиcлений	
	input_dynamic_pri.xml - исправленный входной XML-файл задания.

В моем задании параметры динамической схемы планирования были следующие:
	— прерывание задач — разрешено;
	— схема с явным заданием приоритетов;
	— при равных приоритетах, выбирается более ранняя задача в файле;
	— большему числу приоритета соотвествует больший приоритет;
	— если задача не вошла в свой период, выдается сообщение об ошибке, 
	  но проверка продолжается для остальных периодов.

Ключи компиляции:
	g++ -std=c++17 -Wall tasker.cpp -o tasker

Ключи запуска
	./tasker file_name

Программа написана на языке С++17 (стандарт 17-го года)
с использованием стандартной библиотеки.

Во входном файле, присланном с программой, используется некорректный для стандартной обработки данных символ '”'.Насколько известно, он не является ASCII символом, поэтому в используемых мной Unix-средах обрабатывается некорректно и заменен на ASCII-аналог ' " '
В данном архиве приложен исправленный файл "input_dynamic_pri.xml".
